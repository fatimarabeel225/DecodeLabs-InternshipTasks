# Heart Disease Classification Using AI

## DecodeLabs Internship - Project 2

### Objective

Develop a supervised machine learning classification model to predict whether a patient has heart disease based on medical attributes.

### Dataset

The dataset contains **1025 patient records** with 13 medical features:

- Age
- Sex
- Chest Pain Type
- Resting Blood Pressure
- Cholesterol
- Fasting Blood Sugar
- Resting ECG
- Maximum Heart Rate
- Exercise Induced Angina
- ST Depression
- Slope
- Number of Major Vessels
- Thalassemia

Target Variable:

- `target`
  - 0 → No Heart Disease
  - 1 → Heart Disease Present

### Algorithm Used

**Random Forest Classifier**

### Libraries

- Pandas
- NumPy
- Scikit-learn
- Matplotlib

### Steps

1. Load and analyze dataset
2. Check missing values
3. Prepare features and target variable
4. Split dataset into training and testing sets
5. Train Random Forest model
6. Make predictions
7. Evaluate model performance using metrics

### Model Performance

**Accuracy: 92.20%**

Evaluation Metrics:

- Accuracy Score
- Classification Report
- Confusion Matrix

### Result

The trained AI model successfully classifies heart disease cases based on patient medical information with an accuracy of **92.20%**.
